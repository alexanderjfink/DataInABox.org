The potential of open data is just beginning to be explored. Every day, activists, organizers, NGOs, and governments are demonstrating new ways that open data can be linked, mashed-up, and remixed to build useful, creative, and innovative tools.

Data is everywhere. Everything from Tweets to the personal information social workers collect about clients is "data." However, not all data is public. And not all public data is open. Open data is public data that is also free to use and distribute, and easily accessible. Because of the Open Data Movement, open data is prolific, and growing by the day! New policies, as well as a cultural shift toward greater openness and transparency, are pushing more and more data into open formats.

  • datahub.io
  • datacatalogs.org
  • data.gov
  • opendata.stackexchange.com

Using this kind of data is complicated! No longer do we have to send out a survey, collect data, and analyze it using complex and expensive data analysis software. There's already so much data out there, and as many projects have demonstrated, some very cool projects can be made with the Data already available!
